 Ephoros – Página de Divulgação

A **Ephoros** é uma página web moderna e responsiva desenvolvida apenas com **HTML** e **CSS**, criada para apresentar e divulgar uma aplicação de gestão de estacionamentos.



## Documentação
TAP | Design | Requisitos | Estrutura do Projeto


## Instalação

Como este é um projeto apenas em HTML e CSS, nenhuma instalação via npm é necessária.

Você pode clonar o repositório e abrir o arquivo `index.html` diretamente no navegador.

## Rodando localmente

### Clone o projeto

```bash
git clone https://github.com/Alveszx1/landing-page-lima-projetointegrador
```
## Entre no diretório do projeto

```bash
cd ephoros
```
## Abra o projeto

Abra o arquivo:

```bash
index.html
```

## Variáveis de Ambiente

Este projeto não utiliza variáveis de ambiente.



## Rodando os testes
